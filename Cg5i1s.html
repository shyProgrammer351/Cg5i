문제

A양과 B군이 가위바위보를 선택하면 승패를 출력하는 문제입니다 실행순서를 클릭하세요 
A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 3
보를 냈습니다.
A양이 이겼습니다.

	#include <stdio.h>


	typedef enum { SCISSOR = 1, ROCK = 2, PAPER = 3 } RPSGame;

		int player = ROCK;

		scanf( "%d", &player );


		switch( player ) {

		case SCISSOR:

			printf( "가위를 냈습니다.\n" );

			break;

		case ROCK:

			printf( "바위를 냈습니다.\n" );

			break;

		case PAPER:

			printf( "보를 냈습니다.\n" );

		}


		return player;

F1e	}

		return

				|| ( ( playerA == PAPER ) && ( playerB == ROCK ) ) )

			? TRUE : FALSE;
