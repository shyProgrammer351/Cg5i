문제

캐릭터가 한 칸 이동해서 방패([), 칼(/), 창(->), 출구(문) 중 하나를 선택하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
      / 
  [   옷   문
      -> 
WASD와 [Enter]를 입력하세요: a
      / 
 옷]       문
      ->


      / 
  [   옷   문
      -> 
WASD와 [Enter]를 입력하세요: W
      옷/ 
  [   옷   문
      ->


      / 
  [   옷   문
      -> 
WASD와 [Enter]를 입력하세요: s
      / 
  [        문
      옷->

알고리즘

프로그램시작 
   캐릭터의 초기 위치를 출력 
   이동 방향 입력 받기 
   이동 결과 출력 
      W는 위로 이동 
      A는 왼쪽으로 이동 
      S는 아래로 이동 
      D는 오른쪽으로 이동 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter15/Direction.c

	#include <stdio.h>

	 

	// 방향 매크로 상수 정의

	#define up 'w'

	#define UP 'W'

	#define down 's'

	#define DOWN 'S'

	#define left 'a'

	#define LEFT 'A'

	#define right 'd'

	#define RIGHT 'D'

	 

	// 프로그램시작 

1	int main() {

2		char direction = UP;

	 

		// 캐릭터의 초기 위치를 출력 

3		printf("      / \n");

4		printf("  [   옷   문\n");

5		printf("      -> \n\n");

		 

		// 이동 방향 입력 받기 

6		printf("방향키(WASD)와 [Enter]를 입력하세요: " );

7		scanf("%c",&direction);

	 

		// 이동 결과 출력 

8		printf(" \033[2;6f  ");

9		switch( direction ) {

			// W는 위로 이동 

10			case UP: case up:

11				printf(" \033[1;6f 옷/ ");

12				break;

			// A는 왼쪽으로 이동 

13			case LEFT: case left:

14				printf(" \033[2;1f 옷] ");

15				break;

			// S는 아래로 이동 

16			case DOWN: case down:

17				printf(" \033[3;6f 옷-> ");

18				break;

			// D는 오른쪽으로 이동 

19			case RIGHT: case right:

20				printf(" \033[1;1f              \n");

21				printf("   Success!!   \n");

22				printf("              ");

		}

	 

		// 프로그램 종료 

23		return 0;

	}
