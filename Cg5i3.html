문제

반지름을 입력받아 원둘레와 원면적을 구하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
반지름을 입력하세요 : 3
반지름 3.00인 원둘레는 18.85이고 원면적은 28.27입니다.
반지름 ( 3.00 + 1.00 )인 원둘레는 25.13이고 원면적은 50.27입니다.


반지름을 입력하세요 : 2
반지름 2.00인 원둘레는 12.57이고 원면적은 12.57입니다.
반지름 ( 2.00 + 1.00 )인 원둘레는 18.85이고 원면적은 28.27입니다.


반지름을 입력하세요 : 1
반지름 1.00인 원둘레는 6.28이고 원면적은 3.14입니다.
반지름 ( 1.00 + 1.00 )인 원둘레는 12.57이고 원면적은 12.57입니다.

알고리즘

프로그램시작 
   반지름을 입력 
   주어진 반지름으로 원둘레 및 원면적 계산 
      원둘레 및 원면적 출력 
   (반지름+1)으로 원둘레 및 원면적 계산 
      원둘레 및 원면적 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter15/Circle.c

	#include <stdio.h>

 

	// 매크로 상수 및 매크로 함수 정의

	#define PI 3.141592

1	#define CIRCUMFERENCE( r )  ( 2 * PI * ( r ) )

2	#define CIRCULARAREA( r ) ( PI * ( r ) * ( r ) )

	 

	// 프로그램시작 

3	int main() {

		double radius = 0.0;

		double circumference = 0.0;

4		double circularArea = 0.0;

 

		// 반지름을 입력 

		printf("반지름을 입력하세요 : ");

5		scanf("%lf", &radius );

 

		// 주어진 반지름으로 원둘레 및 원면적 계산 

6		circumference =

7			 	CIRCUMFERENCE( radius );

8		circularArea =

9			 	CIRCULARAREA( radius );

 

		// 원둘레 및 원면적 출력 

10		printf("반지름 %.2f인 원둘레는 %.2f이고 원면적은 %.2f입니다.",

						radius, circumference, circularArea );

 

		// (반지름+1)으로 원둘레 및 원면적 계산 

11		circumference =

12			 	CIRCUMFERENCE( radius + 1 );

13		circularArea =

14			 	CIRCULARAREA( radius + 1 );

 

		// 원둘레 및 원면적 출력 

15		printf("반지름 ( %.2f + 1.00 )인 원둘레는 %.2f이고 원면적은 %.2f입니다.",

						radius, circumference, circularArea);

	 

		// 프로그램 종료 

16		return 0;

	}
