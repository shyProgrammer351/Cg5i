문제

구슬을 3개 컵 중에서 어디에 숨겼는지 사용자가 맞추는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
  ___    ___    ___  
 |   |  |   |  |   | 
 | 1 |  | 2 |  | 3 | 
   O 
 찾았다!
 1, 2, 3 중에서 구슬[O]을 숨긴 컵을 찾으세요 : 1
 [DEBUG] 구슬 위치 1


  ___    ___    ___  
 |   |  |   |  |   | 
 | 1 |  | 2 |  | 3 | 
                 O 
        놓쳤다!
 1, 2, 3 중에서 구슬[O]을 숨긴 컵을 찾으세요 : 2
 [DEBUG] 구슬 위치 3


  ___    ___    ___  
 |   |  |   |  |   | 
 | 1 |  | 2 |  | 3 | 
          O 
               놓쳤다!
 1, 2, 3 중에서 구슬[O]을 숨긴 컵을 찾으세요 : 3
 [DEBUG] 구슬 위치 2
