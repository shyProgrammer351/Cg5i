문제

A양과 B군이 가위바위보를 선택하면 승패를 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 3
보를 냈습니다.
A양이 이겼습니다.

보를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
B군이 이겼습니다. 


A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
A양과 B군이 비겼습니다.

코드 빈칸 채우기

#include <stdio.h> 
 
  { SCISSOR = 1, ROCK = 2, PAPER = 3 } RPSGame; 
  { TRUE = 1, FALSE = 0 } Boolean; 
 
RPSGame getRockPaperScissors() { 
	printf( "가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: " ); 
	int player = ROCK; 
	scanf( "%d", &player );  
 
	( player ) { 
	 SCISSOR: 
		printf( "가위를 냈습니다.\n" );  
		; 
	 ROCK:	 
		printf( "바위를 냈습니다.\n" );  
		; 
	 PAPER: 
		printf( "보를 냈습니다.\n" );  
	} 
 
	 player;									 
} 
	 
Boolean DoesPlayerAWin( RPSGame playerA, RPSGame playerB ) { 
	return 
		( ( ( playerA  SCISSOR ) && ( playerB  PAPER ) ) 			 
			|| ( ( playerA  ROCK ) && ( playerB  SCISSOR ) ) 
			|| ( ( playerA  PAPER ) && ( playerB  ROCK ) ) )  
		 TRUE : FALSE; 
} 
 
int main() { 
	printf( "A양, " ); 
	RPSGame playerA = 
	          getRockPaperScissors(); 
	printf( "B군, " ); 
	RPSGame playerB =  
	          getRockPaperScissors(); 
 
	if ( playerA == playerB ) 
		printf( "\nA양과 B군이 비겼습니다.\n" );  
	else if ( DoesPlayerAWin( playerA, playerB )  
	                                             == TRUE ) 
		printf( "\nA양이 이겼습니다.\n" );  
	else  
		printf( "\nB군이 이겼습니다.\n" );  
 
	 0;									 
}
