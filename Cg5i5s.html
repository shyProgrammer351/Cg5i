문제

우리반 학생들의 평균 점수를 계산하는 문제입니다 실행순서를 클릭하세요 
우리반 학생수를 입력하세요: 3
점수를 입력하세요: 90
점수를 입력하세요: 89
점수를 입력하세요: 86
우리반 학생들의 평균 점수는 88.33입니다

프로그램 코드

	#include <stdio.h>

	#include <stdlib.h>

		

1	int main() {

		printf( "우리반 학생수를 입력하세요: " );

		int count = 0;

2		scanf( "%d", &count );


3		int* scores =

4		              (int*)malloc( sizeof(int) * count );

5		for( int index = 0; index < count; index++ ) {

			printf( "점수를 입력하세요: " );

6			scanf( "%d", &scores[index] );

		}

		

		int sum = 0;

		for( int index = 0; index < count; index++ ) {

			sum += scores[index];

		}

		

7		printf( "우리반 학생들의 평균 점수는 %.2f입니다", (double)sum / count );

		

8		free( scores );

		

9		return 0;

	}
